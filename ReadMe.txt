ReadMe
======

These are settings for the text editor Vim

On Windows
----------
In C:\Users\$USERNAME$\
Create a folder 'vimfiles', which contains the repository

Make your _gvimrc/.gvimrc the following line:
source ../vimfiles/gvimrc

Make your _vimrc/.vimrc the following line:
source ../vimfiles/vimrc

On Linux
--------
1. Extract these files to $HOME/.vim/
2. Edit your $HOME/.vimrc to the following line:
source $HOME/.vim/vimrc
3. Edit your $HOME/.vim/
4. append to $HOME/.bashrc
export TERM=xterm-256color
